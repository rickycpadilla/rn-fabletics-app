# Prototype Fabletics Mobile App
### Setup
NOTE: You should have the latest version of Xcode and React Native CLI installed.
1. Clone this repo
2. `$ cd rn-fabletics-app`
3. `$ npm install`
4. `$ react-native link`
5. `$ react-native run-ios`
